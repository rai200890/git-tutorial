# Instalação

## Debian

```shell
sudo apt update && apt install git-all
```

## Fedora

### Versões mais novas

```shell
sudo dnf install git
```

### Versões mais antigas

```shell
yum install git
```

### Manjaro

```shell
pacman -S git
```

### OSX

```shell
brew install git
```
